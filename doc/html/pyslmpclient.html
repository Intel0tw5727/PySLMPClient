class pyslmpclient.SLMPClient(addr, port=5000, binary=True, ver=4, tcp=False)¶

SLMPによりPLCとやり取りする

パラメータ

  • addr (str) -- 接続するPLCのIPアドレス

  • port (int) -- 接続先のポート番号

  • binary (bool) -- 交信コードとしてバイナリを使用するかどうか

  • ver (int) --

    使用するフレームのバージョン 4 or 3

    CC-Link IE側の用語で行くと4EフレームがMT型、3EフレームがST型

  • tcp (bool) -- TCPで通信するかどうか

コンテキストマネージャに対応しているので open() / close() の代わりに with 文 が使用できる。

target = None¶

通信対象(接続先と通信対象は別個に指定する)

Type

pyslmpclient.util.Target

logger = None¶

モジュールで使用するロガー

Type

logging.Logger

通信の開始

必ず close() とセットで使用する

close()¶

通信終了

必ず open() とセットで使用する
